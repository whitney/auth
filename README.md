sisyphus
========
Real-time push-notification service.

## Running Locally
```
$ ./bin/build
$ ./bin/run
```

## Deploying (new) Code
1.  commit code and push to github
2.  ```./bin/deploy```

## Scaling processes
[web](https://dashboard.heroku.com/apps/days-sisyphus/resources)

## Monitoring
Tailing logs:
```$ heroku logs --tail```
